Help

Textures Sets the block texture. A value of auto will automatically set the texture from the mod ID and the selected block ID. When set to any other value, place the textures in lib/textures/block/ with a filename of [texture].png.

Hardness Controls block breaking time. See the table below for reference values. Value roughly corresponds to five times less than the time in seconds it takes to mine the block by hand.

Block Grass Glowstone Sand Sign Stone Ore Anvil Obsidian Bedrock
Value 0 0.3 0.5 1 1.5 3 5 50 18000000

Resistance Blast resistance. See the table below for reference values.

Block Flower Grass Piston Bamboo Wood Ore Stone Obsidian Bedrock
Value 0 0.3 0.5 1 2 3 6 1200 3600000

Map type Map display type. Controls the colour of this block on a map.

Inventory tab If not set to a predefined value, a new custom inventory tab is created.
